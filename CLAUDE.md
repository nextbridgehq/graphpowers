# CLAUDE.md

This project's shared agent conventions live in [AGENTS.md](AGENTS.md) —
read it before making any changes. It covers changelog format, commit
conventions (incl. no AI co-author attribution), **never pushing to
the remote or opening a PR unless explicitly instructed to for that
specific task**, which files are release-please-owned, graph relation
semantics (`IMPACT_RELATIONS`), the bridge-vs-skills architecture
split, testing conventions, the zero-dependency constraint, the CLI
exit-code contract, and more.
